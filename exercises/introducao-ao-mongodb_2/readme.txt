Instruções para importar arquivo "superheroes.json"

- Abrir o terminal na pasta onde se encontra o arquivo "superheroes.json";
- Executar o comando: mongoimport --db class --collection superheroes "superheroes.json".

Instruções para importar o arquivo "exercise-filter-operators.js"

- Abrir o terminal;
- Digitar o comando: mongo;
- Copiar o conteúdo do arquivo "exercise-filter-operators.js";
- Colar o conteúdo no terminal;
- Apertar "Enter";

Instruções para importar arquivo "books.json"

- Abrir o terminal na pasta onde se encontra o arquivo "books.json";
- Executar o comando: mongoimport --db sample --collection books "books.json".

Instruções para importar o arquivo "bios.js"

- Abrir o terminal;
- Digitar o comando: mongo;
- Copiar o conteúdo do arquivo "bios.js";
- Colar o conteúdo no terminal;
- Apertar "Enter";
